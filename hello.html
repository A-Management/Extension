FAQ

How do I open this window with a keyboard shortcut?

1. Go to "[browser]://extensions/shortcuts". Example:"edge://extensions/shortcuts"

Under -> "Activate the extension", enter the shortcut. Example: Alt+E



How can I change the keyboard shortcuts?

1. Go to "[browser]://extensions/shortcuts". Example:"edge://extensions/shortcuts"

Under -> "Activate the extension", enter the shortcut. Example: Alt+E

For the shortcut to work globally (when you aren't using the browser), it has to be ctrl+shft+[0-9] like ctrl+shft+1

Some combinations have had issues working, so if you encounter that try a different one.



How long does the information in stay there?

Should stay there until the extension is removed or perhaps updated



How secure is the info in ?

The information is stored locally in Chrome. When this was written, it is encrypted using AES256 encryption using a key generated via the salted password you enter. Without the password, it is irrecoverable. It is always encrypted when stored.



Is encrypted?

Not currently. It is stored in plaintext
